# flutter-bootcamp

Personal follow-along work for **Angela Yu's "The Complete Flutter Development
Bootcamp with Dart"** (Udemy). This is course practice — each app is built while
following the lessons, for learning purposes only.

## Environment

- macOS (Apple Silicon), Flutter 3.44.2 (stable), Dart 3.12.2
- Targets: Android emulator (Pixel 7, API 35) and iOS Simulator

## Apps

Each folder holds one course project. Drop the generated Flutter project into its
folder and commit as you go.

### Day 1
| Folder | App | Concepts |
|---|---|---|
| `i_am_poor` | I Am Poor | Widgets, `MaterialApp`, `Scaffold`, assets |
| `micard` | MiCard | Layout, `Container`, `Column`, `Card`, `ListTile` |
| `dicee` | Dicee | Stateful widgets, `setState`, buttons |
| `xylophone` | Xylophone | Packages (`audioplayers`), functions, `Expanded` |
| `quizzler` | Quizzler | Classes/objects, lists, conditionals |
| `destini` | Destini | State management, story logic |

### Day 2
| Folder | App | Concepts |
|---|---|---|
| `bmi_calculator` | BMI Calculator | Themes, custom widgets, multi-screen, routing |
| `clima` | Clima | HTTP, JSON, async/await, geolocation, API keys |
| `bitcoin_ticker` | Bitcoin Ticker | REST APIs, dropdowns, platform widgets |
| `todoey` | Todoey | Provider state management, modal sheets |
| `flash_chat` | Flash Chat (stretch) | Firebase auth + Firestore, animations |

## Notes

- API keys (Clima, Bitcoin Ticker) and Firebase config (Flash Chat) are **not**
  committed — see `.gitignore`.
